History:

My very first Java game, this simple variation of Hangman was just my own self-study project that I devised as a way to learn how to do a simple GUI in Swing. I was in the process of learning Java at the time, and I've found that making games is by far the best way to learn a new language or technology.
Motivation:

I was "motivated" by the new love affair I was falling into with Java. That's about it!
