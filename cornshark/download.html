Download Instructions

  1. If you don't have the latest and greatest version of Java, install it by visiting java.com.
  2. Click here to download the latest version of CornShark in ZIP format.
  3. Unzip the downloaded file to the location of your choosing. Just about anywhere is fine; I recommend your desktop for this particular version.
  4. Double-click on the file called CornShark.jar to run the game.
  5. Email me with any feedback you'd like to share, be it positive, negative, or indifferent.
